# Security Policy

## Supported Versions

Currently, Pythinker CLI only provides security support for the latest version.

## Reporting a Vulnerability

Please report a vulnerability via the [mohamed-elkholy95/pythinker-code - Security](https://github.com/mohamed-elkholy95/pythinker-code/security) page, or open an [issue](https://github.com/mohamed-elkholy95/pythinker-code/issues) if it can be published publicly.
